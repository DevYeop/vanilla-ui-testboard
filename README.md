# 🌟 Vanilla Community Platform (바닐라 커뮤니티)

> 학생들을 위한 쉬운 웹 프로젝트! 🎓

## 이 프로젝트가 뭐예요?

**Vanilla Community Platform**은 **"온라인 게시판"**입니다. 마치 학교 게시판에 글을 붙이듯이, 여기서도 글을 쓰고, 읽고, 검색할 수 있어요! 📝

### 쉽게 말하면:
- 📱 **회원가입/로그인**: 카카오톡처럼 계정을 만들고 로그인해요
- ✍️ **게시글**: 여러분의 생각을 글로 쓰고 공유해요
- 🔍 **검색**: "React"처럼 원하는 글을 찾아요
- 🌙 **밤모드/낮모드**: 밤에는 눈이 편하게 어두운 화면으로 바뀌어요

---

## 🏗️ 어떻게 만들어져 있나요?

### 3가지 주요 부분:

```
┌─────────────────────────────────────────┐
│  🖥️ 프론트엔드 (Frontend)               │
│  - React로 만든 화면                    │
│  - 버튼, 입력창, 게시글 카드 등         │
│  - 위치: client/src/                    │
└─────────────────────────────────────────┘
                    ↕️ 통신
┌─────────────────────────────────────────┐
│  🚀 백엔드 (Backend)                    │
│  - Node.js + Express로 만든 서버        │
│  - 회원가입, 로그인, 게시글 관리        │
│  - 위치: server/                        │
└─────────────────────────────────────────┘
                    ↕️ 저장
┌─────────────────────────────────────────┐
│  💾 데이터베이스 (Database)             │
│  - 사용자 정보 (users.json)             │
│  - 게시글 (posts.json)                  │
│  - 위치: db/                            │
└─────────────────────────────────────────┘
```

## 📂 폴더 구조 설명

```
vanilla-community/
├── client/                 # 화면을 만드는 폴더 🖼️
│   └── src/
│       ├── components/     # 작은 부품들 (버튼, 카드 등)
│       └── pages/          # 전체 페이지들 (홈, 검색 등)
│
├── server/                 # 서버 코드 폴더 🚀
│   ├── routes.ts          # API 주소들 (/api/login, /api/posts 등)
│   ├── storage.ts         # 데이터 저장/불러오기 코드
│   └── index.ts           # 서버 시작 코드
│
├── db/                     # 데이터가 저장되는 폴더 💾
│   ├── users.json         # 사용자 정보
│   └── posts.json         # 게시글
│
├── shared/                 # 프론트엔드, 백엔드 둘 다 쓰는 코드
│   └── schema.ts          # 데이터 규칙 (예: 사용자는 반드시 이름이 있어야 함)
│
└── tests/                  # 자동 테스트 코드 🤖
    ├── comprehensive_test.py  # 전체 기능 테스트
    └── utils.py              # 테스트 도우미 코드
```

## 🚀 시작하기

### 1️⃣ 서버 실행하기

```bash
npm run dev
```

이 명령어를 실행하면:
- 📍 `http://localhost:5000` 에서 웹사이트가 열려요
- 자동으로 프론트엔드와 백엔드가 모두 시작됩니다

### 2️⃣ 테스트 실행하기 (선택사항)

```bash
cd tests
python comprehensive_test.py
```

이 코드는 **자동으로 모든 기능을 테스트**합니다! 🤖
- 회원가입 → 로그인 → 게시글 보기 → 검색 → 밤모드 전환
- 마치 사람이 조작하는 것처럼 자동으로 합니다

## 🎯 주요 기능 설명

### 1. 회원가입 & 로그인

**회원가입 과정:**
1. "회원가입" 버튼 클릭
2. 사용자명 입력 (예: "김철수")
3. 비밀번호 입력
4. 가입 완료! ✅

**로그인:**
- 가입한 사용자명과 비밀번호로 로그인
- 로그인하면 "반갑습니다, 김철수님!" 이라고 인사해줍니다 👋

### 2. 게시글 보기

홈페이지에서 최신 게시글 5개를 볼 수 있습니다:
- 📌 제목
- 📝 미리보기 (처음 150글자)
- 👤 작성자
- 📅 작성 날짜

### 3. 게시글 상세보기

게시글을 클릭하면 **전체 내용**을 볼 수 있어요

### 4. 검색 기능

"React"처럼 단어를 입력하면:
- 모든 게시글 중에서 그 단어가 있는 글들만 보여줍니다
- 예: "React 튜토리얼", "React 프로젝트" 등

### 5. 밤모드/낮모드 (다크 모드)

오른쪽 위 버튼을 클릭하면:
- 🌙 밤모드: 어두운 화면 (눈이 편해요)
- ☀️ 낮모드: 밝은 화면 (기본값)

## 💬 API 주소 (백엔드와 통신하는 방법)

프론트엔드가 백엔드에 요청하는 주소들입니다:

| 기능 | 주소 | 방식 | 예시 |
|------|------|------|------|
| 회원가입 | `/api/signup` | POST | 새 계정 만들기 |
| 로그인 | `/api/login` | POST | 계정으로 로그인 |
| 로그아웃 | `/api/logout` | POST | 로그아웃 |
| 현재 사용자 정보 | `/api/me` | GET | 로그인한 사람 정보 가져오기 |
| 게시글 목록 | `/api/posts` | GET | 게시글 5개 가져오기 |
| 특정 게시글 | `/api/posts/:id` | GET | ID가 "1"인 게시글 가져오기 |
| 검색 | `/api/posts/search?q=React` | GET | "React" 검색 |

## 🧪 테스트 코드 이해하기

`comprehensive_test.py` 파일은 **자동 테스트 프로그램**입니다:

```python
# 예시:
# 1. 회원가입하기
# 2. 로그인하기
# 3. 게시글 5개 있는지 확인
# 4. 게시글 1개 클릭해서 상세보기
# 5. 검색 기능 테스트
# 6. 밤모드로 전환
# 7. 로그아웃
# 8. 모든 기능이 정상 작동하는지 확인 ✅
```

**테스트를 사용하는 이유:**
- 코드를 수정할 때마다 모든 기능이 잘 작동하는지 자동으로 확인
- 버그를 빨리 찾을 수 있음
- 믿을 수 있는 코드를 만들 수 있음

## ❓ 자주 묻는 질문 (FAQ)

**Q: 왜 비밀번호가 안 암호화되어 있나요?** 
A: 학생들이 쉽게 이해하도록 간단하게 만들었어요. 실제 서비스에서는 반드시 암호화해야 합니다! 🔐

**Q: 데이터가 어디에 저장되나요?**
A: `db/` 폴더에 JSON 파일로 저장돼요. 서버를 멈추고 다시 시작해도 데이터가 남아있습니다.

**Q: 여러 사람이 동시에 사용해도 되나요?**
A: 네! 다른 친구들도 `localhost:5000`에 접속해서 같이 사용할 수 있어요.

**Q: 모바일에서도 되나요?**
A: 네! 반응형 디자인이라서 휴대폰에서도 잘 보여요. 같은 와이파이에 연결되면 다른 기기에서도 접속 가능합니다.
